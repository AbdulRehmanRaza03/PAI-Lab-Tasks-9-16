Library Assistant Chatbot

A simple web-based chatbot for library information using Flask.

Installation:
pip install -r requirements.txt

Running the Chatbot:
python app.py

Then open your browser and go to: http://localhost:5000

Features:
- Ask about library hours
- Find book location
- Membership information
- Due dates and fines
- WiFi details
- Contact information
- Quiet zones

Example Questions:
"What are your hours?"
"Where are you located?"
"How do I find a book?"
"What is the membership fee?"
"When are books due?"
"How much is the late fine?"
"Do you have WiFi?"
"How do I contact the library?"

No API keys needed. Uses simple pattern matching.
